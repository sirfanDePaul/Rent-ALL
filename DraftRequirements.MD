# 🛍️ Rent-ALL – Local Rental Marketplace

A web-based platform that enables users to rent and lend everyday items locally. Rent-ALL connects people who own underused items with those who need them temporarily, providing a secure and user-friendly rental experience.

---

## 📌 Overview

Rent-ALL allows users to:
- List items for rent
- Browse and search for nearby items
- Communicate with other users
- Manage rental transactions
- Build trust through ratings and reviews

---

## 👥 User Roles

| Role | Description |
|------|------------|
| User | Browses listings without an account |
| Renter | Requests and rents items |
| Lister | Posts items for rent |
| Admin | Oversees platform safety and disputes |

---

## 🚀 Features & User Stories

### 👤 User Accounts and Profiles
- Sign up and log in  
- Profile includes:
  - Name  
  - Email  
  - Password  
  - Age verification  
  - Location (optional)  
  - Profile picture (optional)  
- Edit profile  

**Estimated Time:** 2–4 days

---

### 📦 Create Listing
- Add:
  - Title  
  - Description  
  - Price  
  - Category  
  - Location  
- Upload at least one image  
- Field validation (price must be positive)  
- Listings appear on:
  - Account page  
  - For You page  

**Estimated Time:** 4–6 days

---

### 🔍 Browse & Search
- View listings on:
  - For You page  
  - User profiles  
- Search & filter by:
  - Keyword  
  - Category  
  - Location  
  - Price  
- Click listing for full details  

**Estimated Time:** 6–8 days

---

### 🔄 Rental Transactions
- Select rental dates  
- Send rental requests  
- Lister can approve/reject  
- Track status:
  - Pending  
  - Approved  
  - Completed  
  - Returned  
- View rental history  

**Estimated Time:** 6–8 days

---

### 💬 Chat System
- Messaging between renter and lister  
- Messages tied to listings  
- Messages persist after refresh  
- Notifications for new messages  
- Rental actions handled through chat  

**Estimated Time:** 5–7 days

---

### ⭐ Rating & Reviews
- 1–5 star ratings  
- Written reviews  
- Visible on profiles and listings  
- Only after completed rentals  

**Estimated Time:** 4–6 days

---

### 📧 Email Notifications
- Notifications for:
  - Rental requests  
  - Approval/rejection  
  - New messages  
- Sent reliably and on time  

**Estimated Time:** 2–4 days

---

### 🔐 Trust Rating System
- Trust score based on:
  - Rental/return history  
  - Identification verification  
  - Ratings  
  - Number of transactions  
- Displayed on profiles  

**Estimated Time:** 3–6 days

---

### 🏅 Badge System
- Earn badges based on activity/trust  
- Visible on profile  

**Badge Tiers:**
- Top Lender  
- Reliable Renter  
- Quick Responder  

**Estimated Time:** 3–5 days

---

### 💡 Smart Pricing Engine
- Suggests prices based on:
  - Similar listings  
  - Category  
  - Location  
  - Quality  
- Optional for users  

**Estimated Time:** 4–6 days

---

### 🗺️ Map View
- View listings geographically  
- Use user location  
- Click pins for details  
- Support different regions  

**Estimated Time:** 5–7 days

---

### 🎯 Personalized "For You" Page
- Recommendations based on:
  - Browsing history  
  - Rental history  
  - Location (if enabled)  
- Dynamically updated  

**Estimated Time:** 5–7 days
